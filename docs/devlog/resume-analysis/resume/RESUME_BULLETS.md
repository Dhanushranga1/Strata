## 🎯 USAGE TIPS

### Customizing for Your Resume:
1. **Pick 3-5 bullets** that match the job description keywords
2. **Quantify everything** - use the metrics from `measured_metrics.json`
3. **Lead with action verbs** - Architected, Engineered, Built, Developed, Implemented
4. **Show impact** - Always end with measurable results
5. **Match the role** - Emphasize frontend for frontend roles, backend for backend, etc.

### ATS Optimization:
- Include keywords from job posting (FastAPI, React, PostgreSQL, etc.)
- Use standard section headers
- Avoid special characters or complex formatting
- Include both acronyms and full terms (RAG = Retrieval-Augmented Generation)

### Tailoring by Level:
- **Junior/Mid:** Focus on technical implementation details
- **Senior:** Emphasize architecture decisions and performance optimization
- **Lead/Principal:** Highlight business impact, scalability, and system design
